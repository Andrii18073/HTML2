HTML

HTML - Мова розмітки гіпертексту або HTML є стандартною мовою розмітки для документів, призначених для відображення у веб-браузері. Він визначає значення та структуру веб-вмісту. Цьому часто допомагають такі технології, як каскадні таблиці стилів (CSS) і мови сценаріїв, такі як JavaScript.Веб-браузери отримують HTML-документи з веб-сервера або з локального сховища та передають документи на мультимедійні веб-сторінки. HTML семантично описує структуру веб-сторінки та початково містить ознаки її зовнішнього вигляду.

Елементи HTML є будівельними блоками сторінок HTML. За допомогою конструкцій HTML зображення та інші об’єкти, такі як інтерактивні форми,можуть бути вбудовані у відтворену сторінку. HTML надає засоби для створення структурованих документів, позначаючи структурну семантику тексту, наприклад заголовки, абзаци, списки, посилання , цитати та інші елементи.

Елементи HTML позначаються тегами, написаними за допомогою кутових дужок. Такі теги, як і безпосередньо вводять вміст на сторінку. Інші теги, такі як і оточують і надають інформацію про текст документа та можуть містити теги піделементів. Браузери не відображають теги HTML, але використовують їх для інтерпретації вмісту сторінки.

HTML може вбудовувати програми, написані на мові сценаріїв, наприклад JavaScript, що впливає на поведінку та вміст веб-сторінок. Включення CSS визначає вигляд і макет вмісту. Консорціум World Wide Web Consortium (W3C), колишній супроводжувач HTML і нині супроводжуючий стандарти CSS, заохочував використання CSS замість явного презентаційного HTML з 1997 року. Форма HTML, відома як HTML5, використовується для відображати відео та аудіо, в основному використовуючи елемент разом із JavaScript.

CSS

CSS — це мова таблиць стилів, яка використовується для опису подання документа, написаного на мові розмітки, такій як HTML або XML (включно з діалектами XML, такими як SVG , MathML або XHTML ). CSS є наріжною технологією Всесвітньої павутини, поряд з HTML і JavaScript.

CSS розроблено для того, щоб уможливити розділення вмісту та презентації, включаючи макет, кольори та шрифти. Це розділення може покращити доступність вмісту; забезпечують більшу гнучкість і контроль у специфікації характеристик презентації; дозволити кільком веб-сторінкам використовувати форматування, вказавши відповідний CSS в окремому файлі .css, що зменшує складність і повторення структурного вмісту; і ввімкніть кешування файлу .css, щоб покращити швидкість завантаження сторінок між сторінками, які спільно використовують файл, і його форматування.

Розділення форматування та вмісту також робить можливим представлення тієї самої сторінки розмітки в різних стилях для різних методів візуалізації, наприклад, на екрані, у друкованому вигляді, голосом (через мовний браузер або програму зчитування з екрана) і на основі Брайл. тактильні пристрої. CSS також має правила для альтернативного форматування, якщо доступ до вмісту здійснюється на мобільному пристрої.

Каскадування імен походить від зазначеної схеми пріоритетів, щоб визначити, яке правило стилю застосовується, якщо більше одного правила відповідає певному елементу. Ця каскадна схема пріоритетів є передбачуваною.

Cпецифікації CSS підтримує Консорціум Всесвітньої павутини (W3C). Тип Інтернет-медіа ( тип MIME ) text/cssзареєстровано для використання з CSS відповідно до RFC 2318 (березень 1998 р.). W3C надає безкоштовну службу перевірки CSS для документів CSS.

Крім HTML, інші мови розмітки підтримують використання CSS, зокрема XHTML, звичайний XML, SVG і XUL. CSS також використовується в наборі віджетів GTK.

JavaScript

JavaScript часто скорочений як JS, є мовою програмування, яка є однією з основних технологій Всесвітньої павутини, поряд з HTML і CSS. Станом на 2023 рік 98,7% веб-сайтів використовують JavaScript на стороні клієнта для поведінки веб-сторінок, часто використовуючи сторонні бібліотеки. Усі основні веб-браузери мають спеціальний механізм. JavaScript для виконання коду на пристроях користувачів.

JavaScript — це мова високого рівня, яка часто скомпільована вчасно і відповідає стандарту ECMAScript. Він має динамічну типізацію, об’єктну орієнтацію на основі прототипу та першокласні функції. Це мультипарадигма, яка підтримує керований подіями, функціональний та імперативний стилі програмування. Він має інтерфейси прикладного програмування (API) для роботи з текстом, датами, регулярними виразами, стандартними структурами даних і об’єктною моделлю документа (DOM).

Стандарт ECMAScript не включає жодних засобів вводу/виводу (I/O), таких як мережеві, сховища чи графічні засоби. На практиці веб-браузер або інша система виконання надає JavaScript API для введення-виведення.

Механізми JavaScript спочатку використовувалися лише у веб-браузерах, але зараз є основними компонентами деяких серверів і різноманітних програм. Найпопулярнішою системою виконання для цього використання є Node.js.

Незважаючи на те, що Java і JavaScript схожі за назвою, синтаксисом і відповідними стандартними бібліотеками, ці дві мови відрізняються один від одного та значно відрізняються за дизайном.

Angular

Angular — це платформа та фреймворк для створення односторінкових клієнтських додатків за допомогою HTML та TypeScript. Angular написаний на TypeScript. Він реалізує основні та додаткові функції у вигляді набору бібліотек TypeScript, які ви імпортуєте у свої програми. Архітектура додатку Angular спирається на певні фундаментальні концепції. Основними будівельними блоками фреймворку Angular є компоненти Angular.

Компоненти є основним будівельним блоком для створення програм в Angular. Використовуючи компонентну архітектуру, Angular прагне створити структуру для організації вашого проекту в керовані, добре організовані частини з чіткими обов’язками, щоб ваш код можна було підтримувати та масштабувати.

Компоненти використовують служби, які надають фонову функціональність, не пов'язану безпосередньо з представленнями, наприклад отриманням даних. Такі сервіси можуть бути впроваджені в компоненти як залежності, що робить ваш код модульним, багаторазово використовуваним та ефективним.Компоненти та послуги – це класи, позначені декораторами. Ці декоратори надають метадані, які повідомляють Angular, як їх використовувати.

Apple

Apple Inc. — американська багатонаціональна технологічна компанія зі штаб-квартирою в Купертіно, Каліфорнія. Станом на березень 2023 року Apple є найбільшою компанією світу за ринковою капіталізацією і з 394,3 мільярдами доларів США найбільшою технологічною компанією до 2022 року. Станом на червень 2022 року Apple є четвертим за величиною постачальником персональних комп’ютерів за одиничними продажами; найбільша виробнича компанія за виручкою; і другий за величиною виробник мобільних телефонів у світі. Вважається однією з «Великої п’ятірки» американських компаній інформаційних технологій разом із Alphabet (материнською компанією Google ), Amazon, Meta та Microsoft.

Apple була заснована як Apple Computer Company 1 квітня 1976 року Стівом Возняком, Стівом Джобсом і Рональдом Уейном для розробки та продажу персонального комп’ютера Apple I Возняка. У 1977 році вона була зареєстрована Джобсом і Возняком як Apple Computer, Inc. Другий комп’ютер компанії, Apple II, став бестселером і одним із перших мікрокомп’ютерів масового виробництва. Apple вийшла на біржу в 1980 році і досягла миттєвого фінансового успіху. Компанія розробила комп’ютери з інноваційним графічним інтерфейсом користувача, включно з оригінальним Macintosh 1984 року, про який було оголошено того року в схваленій критиками рекламі під назвою « 1984 ». До 1985 року висока вартість продукції та боротьба за владу між керівниками спричинили проблеми. Возняк пішов з Apple і зайнявся іншими підприємствами, тоді як Джобс пішов у відставку та заснував NeXT, взявши з собою деяких співробітників Apple.

У міру того, як ринок персональних комп’ютерів розширювався та розвивався протягом 1990-х років, Apple втратила значну частку ринку через недорогу дуополію операційної системи Microsoft Windows на ПК-клони на базі Intel (також відомі як « Wintel »). У 1997 році, за кілька тижнів до банкрутства, компанія купила NeXT, щоб вирішити проблему невдалої стратегії операційної системи Apple і залучити Джобса назад у компанію. Протягом наступного десятиліття Джобс повернув Apple до прибутковості за допомогою низки тактик, включаючи представлення iMac, iPod, iPhone та iPad, які отримали схвалення критиків, запуск кампанії « Думай по-іншому » та інших пам’ятних рекламних кампаній, відкриття роздрібної мережі Apple Store, і придбання багатьох компаній для розширення продуктового портфоліо компанії. Коли Джобс пішов у відставку в 2011 році за станом здоров'я і помер через два місяці, його наступником на посаді генерального директора став Тім Кук.

Apple стала першою публічною американською компанією, вартість якої перевищила 1 трильйон доларів у серпні 2018 року, потім – у 2 трильйони доларів у серпні 2020 року та в 3 трильйони доларів у січні 2022 року. У червні 2023 року її оцінювали трохи більше ніж у 3 трильйони доларів. Компанію критикують щодо практики праці її підрядників, екологічної практики та ділової етики, включаючи антиконкурентну практику та джерела матеріалів. Незважаючи на це, компанія має велику кількість прихильників і користується високим рівнем лояльності до бренду. Його також постійно вважали одним із найдорожчих брендів у світі.

Python

Python — це інтерпретована об’єктно-орієнтована мова програмування високого рівня з динамічною семантикою. Його високорівневі вбудовані структури даних у поєднанні з динамічною типізацією та динамічним зв’язуванням роблять його дуже привабливим для швидкої розробки додатків, а також для використання як мови сценаріїв або з’єднувальної мови для з’єднання існуючих компонентів. Простий, легкий для вивчення синтаксис Python підкреслює читабельність і, отже, знижує вартість обслуговування програми. Python підтримує модулі та пакети, що заохочує модульність програми та повторне використання коду. Інтерпретатор Python і велика стандартна бібліотека доступні у вихідному або двійковому вигляді безкоштовно для всіх основних платформ і можуть вільно поширюватися.

Часто програмісти закохуються в Python через підвищену продуктивність, яку він забезпечує. Оскільки етапу компіляції немає, цикл редагування-тестування-налагодження відбувається неймовірно швидко. Налагоджувати програми на Python легко: помилка чи неправильний вхід ніколи не спричинить помилку сегментації. Натомість, коли інтерпретатор виявляє помилку, він викликає виняток. Якщо програма не вловлює виняток, інтерпретатор друкує трасування стека. Налагоджувач рівня вихідного коду дозволяє перевіряти локальні та глобальні змінні, оцінювати довільні вирази, встановлювати контрольні точки, покроково виконувати код по рядках і так далі. Сам налагоджувач написаний на Python, що свідчить про інтроспективну силу Python. З іншого боку, часто найшвидшим способом налагодження програми є додавання кількох операторів друку до джерела: швидкий цикл редагування-тестування-налагодження робить цей простий підхід дуже ефективним.

C++

C ++ - Компілювання, статично типізована мова програмування загального призначення. Підтримує такі парадигми програмування, як процедурне програмування, об'єктно-орієнтоване програмування, узагальнене програмування. Мова має багату стандартну бібліотеку, яка включає поширені контейнери і алгоритми, введення-виведення, регулярні висловлювання, підтримку багатопоточності та інші можливості. C++ поєднує властивості як високорівневих, так і низькорівневих мов. У порівнянні з його попередником мовою C найбільшу увагу приділено підтримці об'єктно-орієнтованого та узагальненого програмування.

C++ широко використовується для розробки програмного забезпечення, будучи однією з найпопулярніших мов програмування. Область його застосування включає створення операційних систем, різноманітних прикладних програм, драйверів пристроїв, додатків для систем, що вбудовуються, високопродуктивних серверів, а також комп'ютерних ігор. Існує безліч реалізацій мови C++ як безкоштовних, так і комерційних і для різних платформ. Наприклад, на платформі x86 це GCC, Clang, Visual C++, Intel C++ Compiler, Embarcadero (Borland) C++ Builder та інші. C++ вплинув інші мови програмування, насамперед Java і C#.

Синтаксис C++ успадкований від мови C. Спочатку одним із принципів розробки було збереження сумісності з C. Проте C++ не є в строгому сенсі надмножиною C; безліч програм, які можуть однаково успішно транслюватися як компіляторами C, і компіляторами C++, досить велика, але включає всі можливі програми на C.

C#

C# - сучасна, об’єктно-орієнтована та безпечна для типів мова програмування. C# дозволяє розробникам створювати багато типів безпечних і надійних програм, які працюють у .NET. C# бере свій початок у сімействі мов C і буде негайно знайомий програмістам C, C++, Java та JavaScript. У цьому огляді представлено огляд основних компонентів мови C# 11 і попередніх версій. Якщо ви хочете вивчити мову на інтерактивних прикладах, спробуйте вступ до підручників C#.

C# — об’єктно-орієнтована, компонентно-орієнтована мова програмування. C# надає мовні конструкції для безпосередньої підтримки цих концепцій, що робить C# природною мовою для створення та використання програмних компонентів. З моменту свого створення C# додав функції для підтримки нових робочих навантажень і новітніх практик проектування програмного забезпечення. За своєю суттю C# є об’єктно-орієнтованою мовою. Ви визначаєте типи та їх поведінку.

C# широко використовується в розробці ігор і надалі буде домінувати. C# інтегрується з Microsoft і тому має велику цільову аудиторію. Такі функції C#, як автоматичне збирання сміття, інтерфейси, об’єктно-орієнтований тощо, роблять C# популярною мовою розробки ігор.

Java

Java — це високорівнева об’єктно-орієнтована мова програмування на основі класів, розроблена таким чином, щоб мати якомога менше залежностей реалізації. Це мова програмування загального призначення, яка дозволяє програмістам написати один раз і запустити будь-де ( WORA - ), тобто скомпільований код Java може працювати на всіх платформах, які підтримують Java, без необхідності повторної компіляції. Програми Java зазвичай скомпільовані в байт-код , який може працювати на будь-якій віртуальній машині Java (JVM) незалежно від базової архітектури комп’ютера . Синтаксис Java подібний до C і C++, але має менше можливостей низького рівня , ніж обидва з них. Середовище виконання Java надає динамічні можливості (такі як відображення та модифікація коду середовища виконання), які зазвичай недоступні в традиційних скомпільованих мовах.

Java набула популярності незабаром після свого випуску і з тих пір є дуже популярною мовою програмування. Java була третьою за популярністю мовою програмування у 2022 році за даними GitHub, а станом на жовтень 2023 року вона посідає четверте місце за індексом TIOBE. Незважаючи на те, що Java все ще користується великою популярністю, останніми роками спостерігається поступове зниження використання Java, а інші мови, які використовують JVM, набувають популярності.

Java була спочатку розроблена Джеймсом Гослінгом у Sun Microsystems. Він був випущений у травні 1995 року як основний компонент платформи Java Sun Microsystems . Оригінальна та еталонна реалізація Java- компіляторів, віртуальних машин і бібліотек класів спочатку були випущені компанією Sun за власними ліцензіями. Станом на травень 2007 року, відповідно до специфікацій Java Community Process, Sun переліцензувала більшість своїх технологій Java за ліцензією GPL-2.0. Oracle пропонує власну віртуальну машину HotSpot Java Virtual Machine, однак офіційною еталонною реалізацією є OpenJDK JVM, яка є безкоштовним програмним забезпеченням з відкритим вихідним кодом і використовується більшістю розробників, і є JVM за замовчуванням для майже всіх дистрибутивів Linux.

Станом на вересень 2023 року Java 21 є останньою версією, яка також є версією довгострокової підтримки (LTS). Java 8, 11 і 17 — це попередні версії LTS, які все ще офіційно підтримуються.

Php

PHP — це мова сценаріїв загального призначення, призначена для веб-розробки. Спочатку він був створений датсько-канадським програмістом Расмусом Лердорфом у 1993 році та випущений у 1995 році. Довідкову реалізацію PHP зараз розробляє PHP Group. Спочатку PHP був абревіатурою Personal Home Page, але тепер він означає рекурсивний ініціалізм PHP: Hypertext Preprocessor.

PHP-код зазвичай обробляється на веб-сервері інтерпретатором PHP, реалізованим у вигляді модуля, демона або виконуваного файлу Common Gateway Interface (CGI). На веб-сервері результат інтерпретованого та виконаного PHP-коду, який може бути будь-яким типом даних, як-от згенерований HTML або дані двійкового зображення, утворить повну або часткову відповідь HTTP. Існують різноманітні системи веб-шаблонів, системи керування веб-вмістом і веб-фреймворки, які можна використовувати для оркестрування або сприяння генерації відповіді. Крім того, PHP можна використовувати для багатьох завдань програмування поза веб-контекстом, таких як автономні графічні програми та роботизоване управління безпілотниками. Код PHP також можна виконувати безпосередньо з командного рядка.

Стандартний інтерпретатор PHP на базі Zend Engine є безкоштовним програмним забезпеченням, випущеним за ліцензією PHP. PHP був широко перенесений і може бути розгорнутий на більшості веб-серверів на різноманітних операційних системах і платформах.

Мова PHP розвивалася без офіційної письмової специфікації чи стандарту до 2014 року, причому початкова реалізація діяла як стандарт де-факто, якому прагнули слідувати інші реалізації. З 2014 року тривала робота над створенням офіційної специфікації PHP.

W3Techs повідомляє, що станом на вересень 2023 року «PHP використовується на 77,0% усіх веб-сайтів, мова серверного програмування яких ми знаємо». [19] Також повідомляється, що лише 16,7% користувачів PHP використовують версії 8.x, які наразі підтримуються. Більшість із них використовує непідтримуваний PHP 7, точніше PHP 7.4, тоді як 20% використовують PHP 5 (трохи більше, ніж підтримується PHP 8.x), який не підтримується виправленнями безпеки та, як відомо, має серйозні вразливості безпеки. Швидше за все, це вказує на те, що ці веб-сайти є вразливими, ті, що використовують 5.x, хоча це не обов’язково вказує на те, що для тих, хто використовує 7.x, оскільки дистрибутиви Linux, такі як Ubuntu і Debian, надають комерційну підтримку та можуть надавати виправлення безпеки для деякі непідтримувані версії 7.x, наприклад 7.4 PHP.
